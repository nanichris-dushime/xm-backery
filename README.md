## Frontend (Products CRUD)
Open:
- http://localhost:4000/

This page lets you perform CRUD for products using the API.

### Steps
1. Start the backend:
   ```bash
   npm i
   cp .env.example .env
   nodemon src/index.js
   ```
2. Paste your JWT in the frontend "JWT token" field.
3. Create / Update / Delete products.
